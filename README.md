# LeWaterMaker - WordPress图片水印插件

## 插件简介

LeWaterMaker是一款功能强大的WordPress图片水印插件，能够自动为上传的图片添加平铺水印效果。插件采用模块化设计，代码结构清晰，易于维护和扩展。

## 主要功能

- ✅ **自动水印添加**：上传图片时自动添加水印
- ✅ **水印模式选择**：支持平铺模式和单个居中模式
- ✅ **丰富的自定义选项**：字体、大小、颜色、透明度、旋转角度等
- ✅ **智能尺寸检测**：只对符合尺寸要求的图片添加水印
- ✅ **WordPress原生界面**：采用WordPress自带样式设计
- ✅ **响应式设计**：支持移动端和桌面端
- ✅ **完整的生命周期管理**：激活、停用、卸载功能

## 系统要求

- WordPress 5.0 或更高版本
- PHP 7.4 或更高版本
- GD扩展（用于图片处理）

## 安装方法

### 方法一：手动安装

1. 下载插件文件
2. 解压到WordPress的`wp-content/plugins/`目录
3. 在WordPress后台启用插件
4. 进入"设置 > LeWaterMaker"进行配置

### 方法二：通过FTP上传

1. 将插件文件夹上传到`wp-content/plugins/`目录
2. 在WordPress后台启用插件
3. 配置插件设置

## 配置说明

### 基本设置

- **启用水印**：开启/关闭水印功能
- **水印文字**：设置要显示的水印文字内容
- **水印模式**：选择平铺模式或单个居中模式
- **字体**：选择水印文字的字体样式（字体文件位于/assets/fonts目录中）
- **字体大小**：设置水印文字的大小（8-72px）
- **透明度**：设置水印文字的透明度（10%-100%）
- **旋转角度**：设置水印文字的旋转角度（-180°到180°）
- **文本间距**：设置水印文字之间的间距（50-300px）
- **文字颜色**：选择水印文字的颜色
- **最小图片宽度**：只有超过此宽度的图片才会添加水印
- **最小图片高度**：只有超过此高度的图片才会添加水印

### 使用建议

- 建议将透明度设置为30%-70%之间，既不影响图片观看又能起到保护作用
- 旋转角度建议设置为-45°或45°，这样水印效果更美观
- 文本间距根据图片尺寸和水印文字大小进行调整

## 常见问题

### Q: 水印没有显示怎么办？
A: 请检查以下几点：
1. 确认已启用水印功能
2. 检查图片尺寸是否满足最小要求
3. 确认服务器支持GD扩展
4. 检查文件权限设置

### Q: 如何调整水印密度？
A: 通过调整"文本间距"参数来控制水印密度，数值越小密度越大。

### Q: 支持哪些图片格式？
A: 支持JPG、PNG、GIF格式的图片。

### Q: 水印会影响图片质量吗？
A: 插件会尽量保持原图质量，但建议使用高质量原图以获得最佳效果。
